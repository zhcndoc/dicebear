# 首字母

Initials 是一种基于文本的矢量头像样式，会在纯色方形背景上居中渲染一个或两个大字母：经典的字母组合占位符。它具有确定性且易于阅读，因此当用户尚未上传头像时，是一种实用的替代方案。
