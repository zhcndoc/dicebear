# Toon Head

Toon Head 是一种插画风格的矢量头像样式，呈现半身角色肖像，
具有自然的人体比例、多样的肤色、发型和面部毛发。
生成具有动画剧集风格的富有表现力的 SVG 个人资料图标，非常适合
社区应用和社交产品。
