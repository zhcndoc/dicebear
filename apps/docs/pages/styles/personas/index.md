# Personas

Personas 是一种扁平设计的矢量头像风格，采用半身人物形象，带有
细腻的肤色阴影、不同的发型、胡须和眼镜，穿着纯色上衣。
可生成覆盖广泛年龄段和外貌特征的 SVG 个人资料图标。
