# Shape Grid

Shape Grid 是一种抽象的矢量头像样式，它将四个简单形状
（正方形、圆形、三角形和六边形）排列在带有色调的
背景上的 2×2 网格中。生成有趣的 SVG 个人资料图标，既可用作头像占位符，
也可用作装饰性的用户标识。
