# Shape Grid

Shape Grid 是一种抽象的矢量头像样式，它将彩色几何
图块排列成紧凑的网格。生成富有趣味、独特且可自定义的 SVG 个人资料
图标——非常适合作为头像占位符或装饰性的用户标识。
