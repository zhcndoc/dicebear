# Open Peeps

Open Peeps 是一种手绘矢量头像风格，呈现多样的半身角色，
带有草图般的轮廓、可自由搭配的发型、表情和配饰。
可为社区平台、营销页面和社交应用生成友好的 SVG 个人资料图标。
