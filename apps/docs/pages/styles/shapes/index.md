# Shapes

Shapes 是一种抽象的矢量头像样式，由两到三个大型几何图形（方形、圆形和三角形）组成，这些图形叠放并旋转在彩色背景上。生成简约的 SVG 个人资料图标，可用作头像占位符或抽象的用户标识。
