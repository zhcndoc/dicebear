# Initial Face

Initial Face 是一种友好的矢量头像风格，将一双富有表情的眼睛与一个彩色方块上的醒目首字母相结合——半是表情符号，半是字母组合。生成亲切的 SVG 个人资料图标，即使在很小的尺寸下也能显得很有个性，非常适合用户列表和个人资料占位符。
