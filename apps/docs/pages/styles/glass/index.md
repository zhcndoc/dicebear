# Glass

Glass 是一种极简抽象的矢量头像风格，以平滑的颜色
渐变呈现，带有微妙的玻璃质感光泽且不显示任何可见形状。生成 SVG 个人资料
图标，可作为低调的占位符或背景，为精致的
界面增添质感。
