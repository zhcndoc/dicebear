# 趣味表情

趣味表情是一种受表情符号启发的矢量头像风格，它在纯色方块上放置一个单独的
富有表现力的面孔（眼睛、嘴巴，以及偶尔出现的配饰，如太阳镜或
泪水）。可即时生成易于辨识的聊天应用和社交网络个人资料图标。
