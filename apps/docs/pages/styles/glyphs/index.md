# Glyphs

Glyphs 是一种极简抽象的矢量头像样式，将平坦的肩部轮廓与一个独特的、类似字形的帽子或头饰组合在一个柔和的带色调圆形中。生成简单的 SVG 个人资料图标，非常适合作为用户列表和聊天侧边栏中的占位符。
