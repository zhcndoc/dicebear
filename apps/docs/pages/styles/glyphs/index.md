# Glyphs

Glyphs 是一种极简的抽象矢量头像风格，它将扁平的肩部轮廓与软色调圆形中的独特字形帽子或头饰相结合。生成干净、身份感较弱的 SVG 个人资料图标，非常适合作为用户列表和聊天侧边栏中的占位符。
