# Glyphs

Glyphs 是一种极简的抽象矢量头像风格，将平坦的肩部剪影与一个独特的、类似字形的帽子或头饰组合在柔和着色的圆形内。生成轻量身份特征的 SVG 头像图标，非常适合作为用户列表和聊天侧边栏中的占位符。
