# Thumbs

Thumbs 是一种俏皮的矢量头像风格，由圆润、拇指形状的角色组成，采用单一纯色，配以简单的眼睛和嘴巴，并置于带有色调的背景上。生成友好的 SVG 个人资料图标，非常适合作为默认用户头像。
