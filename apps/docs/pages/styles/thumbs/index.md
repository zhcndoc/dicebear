# Thumbs

Thumbs 是一种俏皮的矢量头像风格，特色是圆润的大拇指形角色，采用单一纯色，并在有色背景上配有简洁的眼睛和嘴巴。生成迷人、友好的 SVG 个人资料图标，呈现有趣、平易近人的美学风格。
