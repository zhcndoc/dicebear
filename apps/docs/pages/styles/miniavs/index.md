# Miniavs

Miniavs 是一种扁平设计的矢量头像风格，采用粗壮的半身角色，
配有醒目的发型、简洁的表情和色彩丰富的上衣。生成
迷人、友好的 SVG 个人资料图标，即使在较小尺寸下也清晰可读——
非常适合紧凑的用户列表和移动端 UI。
