# 像素艺术

像素艺术是一种低分辨率的矢量头像风格，它会将半身人物渲染为清晰的像素角色，并配以复古发型、眼睛和有色上衣。
生成 8 位风格的 SVG 个人资料图标，在游戏社区和开发者工具中广受欢迎。
