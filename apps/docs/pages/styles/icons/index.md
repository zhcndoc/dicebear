# 图标

Icons 是一种简洁的矢量头像风格，它会将单个 Bootstrap Icons
图形置于带有色调的背景之上。生成清晰、可缩放的个人资料
图标，非常适合仪表板、管理面板和企业工具，在这些场景中，人物
面孔会显得格格不入。
