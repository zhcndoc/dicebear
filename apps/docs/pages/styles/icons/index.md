# 图标

Icons 是一种简洁的矢量头像风格，会在带有色调的背景上放置一个 Bootstrap Icons
图形。生成可缩放的个人资料图标，非常适合
仪表盘、管理面板和企业工具，在这些场景中，人物面孔会显得不太合适。
