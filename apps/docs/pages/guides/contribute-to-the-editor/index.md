# 为编辑器做贡献

[editor.dicebear.com](https://editor.dicebear.com) 上的编辑器是一个
[Vue 3](https://vuejs.org/) + [Vite](https://vitejs.dev/) +
[PrimeVue](https://primevue.org/) 应用。它位于主
[`dicebear/dicebear`](https://github.com/dicebear/dicebear)
monorepo 中的 `apps/editor/` 下。

设置和工作流程说明位于 monorepo 的贡献指南中：

- `dicebear/dicebear/CONTRIBUTING.md` 中的
  [编辑器更改部分](https://github.com/dicebear/dicebear/blob/10.x/CONTRIBUTING.md#editor-changes-appseditor)
