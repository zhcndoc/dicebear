# 我该如何设置性别？

DiceBear 没有单一的 `gender` 开关，但你可以把任何头像塑造成看起来更
阳刚或更阴柔。每个特征都是独立的选项，因此你可以完全掌控：
选择那些符合你想要外观的选项，例如头发或胡须，然后
省略其余部分。

## 查找并应用选项

[Playground](/playground/) 会为每个选项值显示预览，并允许你将它们组合起来，头像会随着你的操作实时更新。每个[头像样式页面](/styles/) 也会列出相同的选项作为静态参考，并附带预览，因此你可以随时查阅。如果你不想编写任何代码，[Editor](https://editor.dicebear.com) 允许你浏览样式并以可视化方式调整选项。

当你知道自己想要哪些选项后，可以将它们作为 [HTTP API 中的查询参数](/how-to-use/http-api/#options) 传入，或者作为 [JS 库](/how-to-use/js-library/) 及其他库中的选项传入。例如，Avataaars 样式允许你通过 `facialHairProbability=0` 关闭胡须：

```
https://api.dicebear.com/10.x/avataaars/svg?seed=Casey&facialHairProbability=0
```

不同样式的选项各不相同，因此请查看你所使用样式对应的页面。

## 分享并复用选项集

如果你整理了一组自己喜欢的选项，可以在我们的 GitHub Discussions 中的
[Show and tell](https://github.com/orgs/dicebear/discussions/categories/show-and-tell)
下分享。这样其他人就可以在你的工作基础上继续构建并将其调整
以满足自己的需求，而你也可以复用其他人已经分享过的组合。

## 为什么没有专门的性别选项

大多数 DiceBear 风格绘制的是一张脸或一个头部，而不是完整的身体。脸部很少有能清楚看出是男性还是女性的特征，因此并不存在一组可靠的特征，可以让 `male`/`female` 开关去切换。最接近例外的是面部毛发，因为它主要长在男性脸上。即便如此，这也只是一个很弱的信号：许多男性会因为个人选择、文化或宗教原因剃净胡须，而且在很多人群中，男性天生面部毛发也不多，所以没有胡子并不能说明什么。

除了面部毛发之外，什么算“男性化”或“女性化”取决于文化和个人喜好。发型、服装或配饰，在不同地方、不同人眼中可能意味着完全不同的东西。内置开关必须为所有人固定一种解释，而这并不适合每个项目或受众。

DiceBear 被世界各地、各种类型的项目使用，因此它不会对男性或女性头像应该长什么样做任何假设。除非风格的设计者有意按性别来设计，否则没有任何选项会绑定到性别。各项选项描述的是头发、眼镜之类的特征，而不是性别。
